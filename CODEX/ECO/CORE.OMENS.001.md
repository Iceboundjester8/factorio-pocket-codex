# CORE.OMENS.001 — Omen / alert framework (radar-like)

**Type:** C  
**Scope:** CORE  
**Applies to:** CORE / CIV

A unified alert layer that surfaces “world pressure” events:
- gatherings
- unrest/uprisings
- hostile activity escalation

## Principles
- Server-ready: minimal tick cost
- Clear UX: few alert types, consistent icons
- Extensible: other mods can register omen sources

**See also:** CIV.EVENTS.001 (when created)
